# Flask E-Portfolio

A responsive personal portfolio web application built using Flask, HTML, CSS, JavaScript, and SQLite, with an AI assistant (vibe coding assistant).

## Features

- Responsive design
- Dark mode
- BMI Calculator API
- Contact form
- Admin dashboard
- SQLite database

## Technologies

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
